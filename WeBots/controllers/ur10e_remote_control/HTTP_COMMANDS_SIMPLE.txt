  СПИСОК HTTP КОМАНД ДЛЯ УПРАВЛЕНИЯ UR10e
═══════════════════════════════════════════════════════════════

Базовый URL: http://localhost:10002

═══════════════════════════════════════════════════════════════
  БАЗОВЫЕ КОМАНДЫ
═══════════════════════════════════════════════════════════════

1. Справка:
   http://localhost:10002/command?cmd=help

2. Статус робота:
   http://localhost:10002/status

3. Остановить все:
   http://localhost:10002/command?cmd=stop

4. Исходное положение:
   http://localhost:10002/command?cmd=home

═══════════════════════════════════════════════════════════════
  УПРАВЛЕНИЕ СКОРОСТЯМИ (vel)
═══════════════════════════════════════════════════════════════

Формат: vel <номер_сустава> <скорость_в_рад/с>

Сустав 1, скорость 0.5:
   http://localhost:10002/command?cmd=vel%201%200.5

Сустав 2, скорость 0.3:
   http://localhost:10002/command?cmd=vel%202%200.3

Сустав 3, скорость 0.4:
   http://localhost:10002/command?cmd=vel%203%200.4

Сустав 4, скорость 0.2:
   http://localhost:10002/command?cmd=vel%204%200.2

Сустав 5, скорость 0.3:
   http://localhost:10002/command?cmd=vel%205%200.3

Сустав 6, скорость 0.25:
   http://localhost:10002/command?cmd=vel%206%200.25

Все суставы одновременно (порядок: 1,2,3,4,5,6):
   http://localhost:10002/command?cmd=vel_all%200.5%200.3%200.4%200.2%200.3%200.25

═══════════════════════════════════════════════════════════════
  УПРАВЛЕНИЕ ПОЗИЦИЯМИ (pos)
═══════════════════════════════════════════════════════════════

Формат: pos <номер_сустава> <позиция_в_радианах>

Сустав 1, 90 градусов (1.57 рад):
   http://localhost:10002/command?cmd=pos%201%201.57

Сустав 2, 45 градусов (0.785 рад):
   http://localhost:10002/command?cmd=pos%202%200.785

Сустав 3, 180 градусов (3.14 рад):
   http://localhost:10002/command?cmd=pos%203%203.14

Все суставы одновременно (порядок: 1,2,3,4,5,6):
   http://localhost:10002/command?cmd=pos_all%200%200.5%200%200%200%200

0°    = 0.000 рад
30°   = 0.524 рад
45°   = 0.785 рад
60°   = 1.047 рад
90°   = 1.571 рад
120°  = 2.094 рад
180°  = 3.142 рад
270°  = 4.712 рад
360°  = 6.283 рад

  ПРИМЕРЫ ПОСЛЕДОВАТЕЛЬНОСТЕЙ
═══════════════════════════════════════════════════════════════

Последовательность 1: Домой → Движение → Стоп
   1. http://localhost:10002/command?cmd=home
   2. http://localhost:10002/command?cmd=vel%201%200.5
   3. http://localhost:10002/command?cmd=stop

Последовательность 2: Вращение нескольких суставов
   1. http://localhost:10002/command?cmd=vel%201%200.5
   2. http://localhost:10002/command?cmd=vel%202%200.3
   3. http://localhost:10002/command?cmd=vel%203%200.4

Последовательность 3: Установка позиций
   1. http://localhost:10002/command?cmd=pos%201%201.57
   2. http://localhost:10002/command?cmd=pos%202%200.785
   3. http://localhost:10002/command?cmd=home

═══════════════════════════════════════════════════════════════
  БЫСТРЫЙ СТАРТ
═══════════════════════════════════════════════════════════════

1. Проверить статус:
   http://localhost:10002/status

2. Вернуть в исходное положение:
   http://localhost:10002/command?cmd=home

3. Начать движение:
   http://localhost:10002/command?cmd=vel%201%200.5

4. Остановить:
   http://localhost:10002/command?cmd=stop

═══════════════════════════════════════════════════════════════
  ВАЖНО
═══════════════════════════════════════════════════════════════

• Пробелы в URL должны быть заменены на %20
• Номера суставов: от 1 до 6
• Скорости: в радианах в секунду (рад/с)
• Позиции: в радианах (рад)
• Убедитесь, что Webots запущен и симуляция активна!
