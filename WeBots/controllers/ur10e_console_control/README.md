# Контроллер консольного управления UR10e

## Как использовать консоль в Webots

### Способ 1: Окно Controller Console (Рекомендуется)

1. **Запустите симуляцию** в Webots
2. **Откройте окно Controller Console**:
   - Оно может открыться автоматически
   - Или найдите его в меню: `Tools` → `Controller Console`
   - Или щелкните правой кнопкой мыши на роботе → `Show Robot Window` → вкладка `Console`
3. **Введите команду** в поле ввода внизу окна консоли
4. **Нажмите Enter** для выполнения команды

### Способ 2: Через файл команд

1. Создайте файл `commands.txt` в папке контроллера:
   ```
   WeBots/controllers/ur10e_console_control/commands.txt
   ```
2. Запишите команду в файл (одна команда на строку)
3. Контроллер автоматически прочитает команду и выполнит её
4. Файл будет очищен после чтения

### Способ 3: Через терминал (если запущен отдельно)

Если контроллер запущен из терминала, просто вводите команды в терминал.

## Доступные команды

### Базовые команды

- `help` - показать справку по всем командам
- `stop` - остановить все моторы
- `status` - показать текущее состояние всех суставов
- `home` - вернуть робота в исходное положение (все суставы в 0)

### Управление скоростью

- `vel <j> <v>` - установить скорость сустава j (1-6) в рад/с
  - Пример: `vel 1 0.5` - вращать сустав 1 со скоростью 0.5 рад/с
  
- `vel_all <v1> <v2> <v3> <v4> <v5> <v6>` - установить скорости всех суставов
  - Пример: `vel_all 0.5 0.3 0.4 0.2 0.3 0.25`

### Управление позицией

- `pos <j> <p>` - установить позицию сустава j в радианах
  - Пример: `pos 1 1.57` - установить сустав 1 в позицию 1.57 рад (90°)
  
- `pos_all <p1> <p2> <p3> <p4> <p5> <p6>` - установить позиции всех суставов
  - Пример: `pos_all 0 0.5 0 0 0 0`

## Примеры использования

### Пример 1: Вращение одного сустава
```
vel 1 0.5
```

### Пример 2: Установка всех скоростей
```
vel_all 0.5 0.3 0.4 0.2 0.3 0.25
```

### Пример 3: Установка позиции сустава
```
pos 1 1.57
```

### Пример 4: Возврат в исходное положение
```
home
```

### Пример 5: Проверка состояния
```
status
```

## Примечания

- Скорости указываются в **радианах в секунду**
- Позиции указываются в **радианах**
- Для перевода градусов в радианы: `радианы = градусы × π / 180`
- Пример: 90° = 1.57 рад, 180° = 3.14 рад

## Устранение проблем

### Команды не выполняются

1. Убедитесь, что окно Controller Console открыто
2. Проверьте, что симуляция запущена
3. Попробуйте использовать файл `commands.txt`

### Робот не двигается

1. Проверьте команду `status` - возможно, моторы не найдены
2. Убедитесь, что используете правильные номера суставов (1-6)
3. Попробуйте команду `home` для проверки
